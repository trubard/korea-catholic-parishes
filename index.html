⛪ 한국 천주교 성당 정보 API

CBCK 주소록에서 매일 자동 수집하는 성당 정보 · 정적 JSON

– 개 성당

엔드포인트

  • data/churches.json — 전체
  • data/index.json — 교구별 요약
  • data/by-diocese/<diocese_id>.json — 교구별

사용 예시

const res = await fetch("./data/churches.json");
const { churches } = await res.json();
